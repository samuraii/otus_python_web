# Инструкция по установке и запуску

Склонировать репозиторий:

```bash
git clone git@github.com:samuraii/otus_python_web_-1.git
```

Чтобы запустить скрипт:

```bash
python count_verbs.py
```

Чтобы запустить тесты:

```bash
python tests.py
```

Результат:

